# Taro-Project

### run project

```
npm install
```

```
npm run dev:h5
```

### build project

```
npm run build:h5
```

#### 目录结构
```bash
├── /dist/           # 项目打包输出目录
├── /config/         # webpack配置目录
├── /src/            # 项目源码目录
│ ├── /assets/       # 静态资源
│ ├── /components/   # 组件目录
│ ├── /pages/        # views目录
│ ├── /utils/        # 工具函数
│ ├── index.html     # 入口文件
│ └── app.less       # 样式文件
│ └── app.js         # app入口
├── package.json     # 项目信息
└── .eslintrc        # Eslint配置
```
